## 🚀 Como Usar

### Opção 1: Abrir Direto no Navegador (Mais Fácil)
1. Duplo clique em `index.html`
2. Pronto! Sistema abre no navegador

### Opção 2: Com Servidor Local (Recomendado)
```bash
# Python
python -m http.server 8000
# Abra http://localhost:8000

# Node.js
npx http-server
# Abra http://localhost:8080
```

### Opção 3: Copiar Arquivos (Produção)
1. Copie os 3 arquivos de código (HTML, CSS, JS)
2. Coloque em seu servidor web
3. Configure o backend conforme necessário

         │ Emite encaminhamento
         ▼
┌─────────────────────────┐
│  REGULASUS - Cadastro │  Registra no sistema
└────────────┬────────────┘
             │ Auditoria
             ▼
┌──────────────────────────┐
│ Médico Regulador - Avalia │  Aprova ou rejeita
└────────────┬─────────────┘
             │ Autorizado
             ▼
┌─────────────────────┐
│ Fila de Espera      │  Paciente aguarda vaga
└────────────┬────────┘
             │ Vaga disponível
             ▼
┌──────────────────────────┐
│  Agendamento             │  Marca data e horário
└────────────┬─────────────┘
             │ Notificação
             ▼
┌──────────────────────────┐
│  Hospital/Ambulatório    │  Paciente realiza consulta
